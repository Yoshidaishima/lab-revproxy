# Proxy with Nginx
## About
Nginx reverse proxy with lets encrypt certbot

## Installation

- Podman || Docker required
## Usage

- Need to run as root
  - Needs sudo permissions to write certs
- Pod will only show up under root
  - Podman is user based

```
# Run
sudo ./run

# Remove
sudo ./rm
```

## Support
- [lets Encrypt](https://letsencrypt.org/getting-started/)
- [Certbot](https://certbot.eff.org/)
